📘 इस्तेमाल कैसे करें (Hindi Usage Guide)

1. नया क्लाइंट जोड़ें

ऊपर "+ New Client" बटन दबाएँ। नाम, कंपनी, ईमेल, फ़ोन, प्लेटफ़ॉर्म (Upwork/Fiverr/Direct/LinkedIn), स्टेटस और प्रायोरिटी भरें। सेव करते ही client card तुरंत दिखेगा।

2. Communication Log करें

किसी भी client card पर क्लिक करें — वहाँ "+ Add Communication" से call, email, message या meeting का record डालें। अगली बार follow-up की तारीख भी वहीं सेट करें — यही तारीख automatically Follow-ups टैब में आ जाएगी।

3. Follow-ups ट्रैक करें

"Follow-ups" टैब खोलें — यहाँ लाल रंग में Overdue (मिस हो चुके) और पीले रंग में जल्द आने वाले follow-ups दिखेंगे। हर दिन एक बार यह टैब ज़रूर चेक करें।

4. Message Templates इस्तेमाल करें

"Message Templates" टैब में client चुनें, भाषा (Hinglish/Hindi/English) चुनें, फिर template card पर क्लिक करें। "Quick Template" से बिना AI के तुरंत message बनेगा, या "AI Se Message Banao" दबाकर ज़्यादा personalized message बनवाएँ — extra context box में project या payment की डिटेल लिखें तो AI उसे भी शामिल करेगा।

5. डेटा सुरक्षित रखें

यह tool आपका सारा डेटा सिर्फ़ आपके browser में (LocalStorage) रखता है — कहीं cloud पर नहीं भेजता। इसलिए "Settings" टैब से हफ़्ते में एक बार "Export" ज़रूर करें ताकि browser cache clear होने पर डेटा न खोए। दूसरे device पर shift करना हो तो वही file "Import" कर लें।

💡 Freelancer Tip

  • हर नए client को "Prospect" स्टेटस से शुरू करें, deal होते ही "Active" में बदलें।
  • High priority उन clients को दें जिनसे payment या ongoing project जुड़ा है।
  • Follow-up date हमेशा तभी set करें जब communication log कर रहे हों — इससे कोई client miss नहीं होगा।
